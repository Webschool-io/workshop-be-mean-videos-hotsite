Como o curso é realizado?

Iniciaremos conhecendo o MongoDb e como utilizá-lo de diversas formas através do Javascript em seu console, após essa base de banco de dados não relacional conheceremos o Node.js, o qual foi o maior divisor de água no mundo do Javascript, após conseguirmos criar um CRUD utilizando Node.js e MongoDb (Mongoose), com rotas em Express avançaremos para o Frontend onde tudo muda e teremos que nos ambientar ao conceito de Single Page Application. Com isso faremos com que o AngularJs requisite os dados de nosso webserver via REST e monte a view corretamente, fazendo um integração completa usando apenas Javascript.
